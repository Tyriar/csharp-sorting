## csharp-sorting

A collection of sorting algorithms written in C# for [Growing with the Web](http://www.growingwiththeweb.com/).

## Dependencies

This project requires [.NET Core](https://www.microsoft.com/net/core).

## Build

```bash
dotnet restore
```

## Test

```bash
dotnet test
```
